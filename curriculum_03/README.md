# curriculum_03 / Git基礎

## 実践演習の進め方

1. このフォルダをGitリポジトリとして初期化する
2. practice.txt を編集してコミット練習をする
3. GitHubにリポジトリを作成してプッシュする

## 手順

```bash
cd curriculum_03
git init
git add practice.txt
git commit -m "feat: Gitの練習を開始"
git remote add origin https://github.com/あなたのユーザー名/python-training.git
git push -u origin main
```
