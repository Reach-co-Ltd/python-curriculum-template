# curriculum_07 / アプリ制作①

## 進め方

1. `todo_app/` の中に `tasks.py` を自分で新規作成する
2. バックログの写経問題を上から順に読みながら `tasks.py` に実装する
3. 各セクションが終わったら `python tasks.py` で動作確認する

## 実行方法
```bash
cd curriculum_07/todo_app
python tasks.py
```

## 提出ファイル
```
curriculum_07/
└── todo_app/
    ├── tasks.py      ← 自分で作成・実装する
    └── tasks.json    ← アプリ起動後に自動で作成される
```
