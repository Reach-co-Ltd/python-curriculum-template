# curriculum_07 / アプリ制作①（ToDoアプリ）

## 提出ファイル構成

```
curriculum_07/
└── shakyou_07/
    └── todo_app/
        ├── tasks.py        ← 自分で新規作成・実装する（メインファイル）
        ├── tasks.json      ← アプリ起動後に自動生成される
        └── template.html   ← HTML出力機能を実装した場合のみ
```

## 進め方
1. `shakyou_07/todo_app/` の中に `tasks.py` を自分で新規作成する
2. カリキュラムの指示に従って実装する
3. `python tasks.py` で動作確認する

## 実行方法
```bash
cd curriculum_07/shakyou_07/todo_app
python tasks.py
```
